# Serverless Motorbike Shop

Serverless to manage the motorbike shop customers

## Quick Start

Install Node Version Manager [NVM](https://github.com/nvm-sh/nvm).

Install node v16

```bash
nvm install v16
nvm use v16
```

install deps

```bash
npm install
```

run example functions

```bash
npm run hello:http
npm run hello:sqs
```

run tests

```bash
npm run test
npm run test:coverage
```
